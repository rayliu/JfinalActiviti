项目简介

一站式Java Web框架, 包含UI+后端+工作流等常用组件, 可使项目快速上线, 圆你创业梦


JFinalActiviti智能开发平台，顾名思义就是结合了JFinal的极速开发优点+Activiti工作流程引擎的一站式Java Web框架。
它可以应用在任何Java Web互联网项目的开发中，尤其适合企业信息管理系统（MIS）、内部办公系统（OA）、企业资源计划系统（ERP） 、客户关系管理系统（CRM）等， 其包含的前端UI和后端主流的整体开发方式，可以显著提高开发效率60%以上，极大降低开发成本。

JFinal_Activiti具有以下特点：
  1. 1.采用热门JFinal极速 WEB + ORM 框架，容易上手
  2. 2.采用热门Activiti工作流框架，轻松配置业务流程

  4. 4.基于RBAC的权限管理实现, 不必烦心如何设计权限分配模块, 拿来即用。
工作流DEMO: 请假流程

注意: demo流程只写了正向流程的所有页面, 拒绝的没写

页面操作步骤:

  • 1. 申请人新建请假单, 保存并提交, 开始流程
  • 2. 部门经理查看待办任务, 看到一个任务待审批
  • 3. 部门经理点击待办任务, 选择审批'同意', 点击完成任务, 此时生成一个新的任务给人事部
  • 4. 人事部文员查看待办任务, 看到一个任务待审批
  • 5. 人事部文员点击待办任务, 选择审批'同意', 点击完成任务, 此时生成一个新的任务给申请人(需销假)
  • 6. 申请人查看待办任务, 看到一个任务待操作
  • 7. 申请人点击待办任务(销假), 点击完成任务, 此时流程结束
  • 8. 申请人在请假单列表中点击"跟踪流程", 可查看流程当前流转状态(红色表示已执行节点)
